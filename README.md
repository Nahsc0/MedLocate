# MedLocate

MedLocate is a Geographic Information System (GIS) map project designed to help users search, filter, and locate medical facilities in Kaduna State, Nigeria. The project aims to provide vital geospatial information to improve access to healthcare services and enhance decision-making in public health.

## Features

- **Search and Filter**: Easily search for medical facilities by name, type, or other criteria.
- **Geospatial Information**: Access key information including:
  - Population density within defined boundaries
  - Facility boundaries represented as polygons
  - Number of patients served by each facility
  - Locations of doctors and medical staff
  - Road networks for easier navigation
  - Proximity analysis for facility access

## Technologies Used

- **GIS Tools**: [QGIS](https://www.qgis.org) for mapping and geospatial analysis.
- **Web Technologies**: [Leaflet](https://leafletjs.com) for interactive maps, HTML, CSS, JavaScript for the front-end.
- **Data Sources**: [OpenStreetMap](https://www.openstreetmap.org) for base map and road data; local health department data for facility details.
